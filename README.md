##  Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/ARCHANA-SENGUNTHAR/table-of-wonders.git
   ```

2. **Navigate to the project folder**

    ```bash
    cd table-of-wonders
    ```

3. **Open *index.html* with Live Server in VS Code for better development experience**
